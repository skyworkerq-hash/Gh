# Папка секретов

Здесь хранятся API-ключи, пароли, токены, логины.

## Правила

- **Эта папка под `.gitignore`.** Всё содержимое, кроме этого `README.md`, **не попадает в git** и не уходит в GitHub.
- Реальные значения лежат в `credentials.json` (см. структуру ниже). Этот файл создаётся локально и не коммитится.
- В облачных сессиях файлы здесь **не переживают перезапуск контейнера** — это сознательная цена за безопасность.
- Долгосрочное хранение для облака — переменные окружения Claude Code on the web (настраиваются в веб-интерфейсе).
- На Mac можно дополнительно использовать macOS Keychain.

## Как я (Claude) с этим работаю

- Я **читаю** `credentials.json` когда мне нужны ключи для задачи.
- Я **никогда** не записываю значения ключей в другие файлы (`claude-vault/`, код, журналы) и не упоминаю их в коммитах.
- В чате значения **не повторяю** даже если их попросишь — только подтверждаю, что нашёл.
- При коммитах смотрю diff на предмет случайных утечек.
