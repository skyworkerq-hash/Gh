# Контекст для Claude

Это репозиторий с **постоянной памятью пользователя** и его проектами. Каждая облачная сессия эфемерна, поэтому всё, что нужно помнить, хранится здесь в git.

## ⚠️ ПЕРВОЕ ДЕЙСТВИЕ В КАЖДОЙ СЕССИИ

**Прочитай эти файлы по порядку, прежде чем отвечать пользователю:**

1. `claude-vault/CLAUDE.md` — инструкции по работе с памятью
2. `claude-vault/knowledge/context.md` — кто пользователь, чем занимается, болевые точки
3. `claude-vault/knowledge/preferences.md` — стиль общения, правила использования скилов/MCP
4. Последний файл в `claude-vault/journal/` (по дате) — на чём остановились в прошлой сессии
5. `claude-vault/projects/*/README.md` — активные проекты (если есть упоминания в журнале)

После загрузки этих файлов можно отвечать пользователю — но **в первом ответе кратко подтверди, что контекст загружен** и напомни, на чём остановились.

## Структура репозитория

```
.
├── CLAUDE.md                ← этот файл
├── .claude/
│   ├── settings.json        ← MCP-серверы, хуки
│   └── hooks/               ← SessionStart хук установки скилов
├── claude-vault/            ← постоянная память Claude
│   ├── CLAUDE.md            ← правила работы с памятью
│   ├── knowledge/           ← контекст + предпочтения
│   ├── projects/            ← активные проекты
│   ├── journal/             ← дневник сессий по датам
│   └── secrets/             ← API-ключи (под .gitignore, кроме README)
├── apps/                    ← готовые приложения
│   └── pdf-editor/          ← веб-редактор PDF с OCR
└── secrets/credentials.json не существует на старте сессии (gitignore)
```

## Язык

Общаемся **на русском**. Подробнее — в `claude-vault/knowledge/preferences.md`.

## Команды-триггеры пользователя

- **«загрузи память»** / **«что помнишь»** — пройди по шагам выше и кратко перескажи где остановились.
- **«обнови память»** — обнови journal/сегодня.md, тронутые projects/*/notes.md, при необходимости knowledge/, закоммить и запушь.
- **«проверь DeepSeek»** — выполни тест API через ключ из `claude-vault/secrets/credentials.json` (если файла нет — попроси у пользователя).

## Важное про секреты

- API-ключи лежат в `claude-vault/secrets/credentials.json` (под `.gitignore`).
- Этот файл **не переживает перезапуск контейнера**. В новой сессии его придётся восстанавливать.
- В чате значения ключей **никогда не повторяй**.
