# Контекст для Claude

Это репо с **постоянной памятью пользователя** в `claude-vault/`. Каждая облачная сессия эфемерна — память хранится в git.

## 🚨 Самое важное

**Пользователь может в любой момент написать `/м` или `/память`** — это slash-команда, которая загружает весь его контекст. После неё ты знаешь кто он, что делаем, где остановились.

**Если пользователь пишет просто «загрузи память», «что помнишь», «привет», или начинает разговор не уточняя задачу** — сначала прочитай эти файлы, потом отвечай:

1. `claude-vault/CLAUDE.md`
2. `claude-vault/knowledge/context.md`
3. `claude-vault/knowledge/preferences.md`
4. Самый свежий файл в `claude-vault/journal/`
5. `claude-vault/projects/*/README.md`

После загрузки кратко подтверди: «Память загружена, мы делали X, остановились на Y».

## Структура

```
.
├── CLAUDE.md                ← этот файл, читается Claude Code автоматически
├── .claude/
│   ├── settings.json        ← MCP-серверы, SessionStart хук
│   ├── hooks/               ← хук установки community-скилов
│   └── commands/            ← /м, /память — загрузка контекста
├── claude-vault/            ← постоянная память
│   ├── CLAUDE.md            ← правила
│   ├── knowledge/           ← кто пользователь, как с ним работать
│   ├── projects/            ← активные проекты
│   ├── journal/             ← дневник сессий по датам
│   └── secrets/             ← API-ключи (под .gitignore)
└── apps/
    └── pdf-editor/          ← веб-редактор PDF с OCR
```

## Язык

Русский. Подробнее — в `claude-vault/knowledge/preferences.md`.

## Главные команды-триггеры

| Пользователь пишет | Ты делаешь |
|---|---|
| `/м` или `/память` или «загрузи память» | Читаешь vault → краткое summary |
| «обнови память» | Обновляешь journal/сегодня, projects/notes, коммитишь, пушишь |
| «проверь DeepSeek» | Тест API через `claude-vault/secrets/credentials.json` |

## Про секреты

API-ключи в `claude-vault/secrets/credentials.json` (под `.gitignore`). Файл **не переживает** новую облачную сессию. Если его нет — попроси пользователя прислать ключ ещё раз.

В чате значения ключей **никогда не повторяй**.
