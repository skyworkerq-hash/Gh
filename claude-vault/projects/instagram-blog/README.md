# Проект: instagram-blog — Управа блога

## Что это
Веб-панель для ведения блога о растениях в Instagram (`@vikamomplants`).
Главная функция (по выбору пользователя) — **банк идей и рубрик**: чтобы не выдумывать темы постов каждый раз.

## Тип
Один HTML-файл (как PDF-редактор), работает в браузере, данные хранятся локально в `localStorage`. Постинг в Instagram — вручную. Без бэкенда, без API.

Файл: `apps/instagram-blog/index.html`.

## Что внутри (MVP от 2026-05-28)
- 14 рубрик под нишу растений (уход, растение недели, ошибки новичков, до/после, размножение, лайфхаки, личное, вопрос-ответ, сезонное, подборки, вредители, декор, покупки, вовлечение).
- ~55 готовых идей постов с подсказками и форматом (Пост / Карусель / Reels / Stories).
- Добавление/редактирование/удаление своих идей.
- Фильтры: по рубрике, по формату, поиск, «только избранное», «скрыть использованные».
- Пометки: ⭐ избранное, ✓ опубликовано.
- Кнопка «🎲 Подкинь идею» — случайная неиспользованная идея.
- Статистика: всего / ждут публикации / опубликовано / в избранном.
- Экспорт/импорт копии (JSON) для бэкапа, «вернуть базовые идеи».

## Решения
- **Только банк идей + рубрики** (не календарь, не автопостинг) — выбор пользователя через AskUserQuestion.
- **Без Instagram API** — автопостинг требует Meta Business, токены, модерацию приложения; для старта избыточно. Постинг вручную.
- **localStorage** — данные на устройстве пользователя, никуда не уходят (приватность).

## Идеи на потом (если попросит)
- Контент-план / календарь.
- Канбан статусов (идея → черновик → готово → опубликовано).
- Генератор подписей и хэштегов.
- AI-подсказки через DeepSeek (идеи/тексты под конкретную тему).
