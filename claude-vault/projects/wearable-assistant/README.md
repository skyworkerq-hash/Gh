# Office Assistant — стационарный AI-хаб на ESP32-P4

> Рабочее название проекта (бывший wearable-assistant).
> Финальное имя ассистента пользователь придумает позже.

## Идея

**Стационарный AI-хаб для офиса:** круглый дисплей с микрофонами,
динамиком и камерой стоит на рабочем столе. Слышит, видит, говорит,
имеет память. Через DeepSeek V4 выполняет команды: помощь по
банковским гарантиям, разбор документов, заметки, напоминания, поиск
информации, тендеры, OCR договоров перед камерой, базовая
автоматизация.

Не носимое. Не Telegram-бот. Не умные часы.

## Финальное железо

**Waveshare ESP32-P4-WIFI6-Touch-LCD-3.4C/4C**
- Артикул AliExpress: **`1005009167855279`**
- Цена: ~$74 (3.4″) / ~$88 (4″)
- Доставка: ~3 недели

**Что внутри:**
- ESP32-P4 RISC-V двухъядерный (новейшее поколение Espressif, 2025)
- ESP32-C6 для Wi-Fi 6 + BT 5 LE
- Круглый IPS LCD 3.4″ 800×800 (или 4″ 720×720), 10-точечный тач,
  optical bonding закалённое стекло, 170° обзор
- **2 цифровых микрофона + ES7210 echo cancellation** — far-field
  распознавание через комнату
- ES8311 audio codec, аудио-выход
- **MIPI-CSI разъём** — для подключения камеры
- 40-pin GPIO как Raspberry Pi
- USB OTG 2.0 HS
- Поддержка DeepSeek заявлена производителем

**Что НЕ идёт в комплекте:**
- ❌ Аккумулятор (не нужен — стационарка, питание от USB-C)
- ❌ Камера (докупаем отдельно)
- ❌ GPS-модуль (опционально)
- ❌ Корпус (плата с открытыми краями; можно докупить или
  напечатать)

## Главное архитектурное решение

**OpenClaw как агент-фреймворк, MimiClaw как прошивка.**
Но MimiClaw пока только под ESP32-S3, не под P4 — поэтому
прошивку под P4 пишем сами с использованием ESP-IDF + ESP-SR
(официальный voice AI стек Espressif).

## Архитектура

```
[Waveshare ESP32-P4 хаб в офисе]
  ├── 2 микрофона + echo cancel → захват голоса (far-field)
  ├── Wi-Fi 6 → передача аудио на сервер
  ├── Динамик + ES8311 → проигрывание ответов
  ├── 3.4″ круглый AMOLED → лицо ассистента, статусы, диалоги
  ├── Камера через MIPI-CSI (SC2336) → визуальный AI
  └── (опц.) GPS NEO-M8N через GPIO → координаты у окна / на выезде

           |
           v (по Wi-Fi 6)

[Сервер на Mac или VPS, OpenClaw + наши надстройки]
  ├── Кастомный STT-pipeline:
  │   └── Whisper (faster-whisper локально или OpenAI API) — русский
  ├── OpenClaw (Node.js):
  │   ├── DeepSeek V4 как кастомный OpenAI-совместимый провайдер
  │   ├── Скилы:
  │   │   ├── обвязки Yandex Taxi / Yandex Maps / Yandex Eda
  │   │   ├── работа с tender-monitor
  │   │   ├── разбор документов (через PDF-редактор)
  │   │   ├── OCR с камеры (визуальный AI через DeepSeek)
  │   │   ├── append_to_obsidian, reminders, etc.
  │   │   └── work_helpers (БГ-специфичные)
  │   └── Telegram-канал (резервный интерфейс)
  ├── Память:
  │   ├── Obsidian-vault (Markdown) — синхронизация через Yandex Disk
  │   └── Chroma (векторная БД) для семантического поиска
  └── TTS:
      └── Yandex SpeechKit (русский, дёшево)
```

## Открытые вопросы

1. **Имя ассистента** — пользователь придумывает.
2. **Сервер — Mac или VPS?** На стадии 1 — Mac. Когда P4 будет
   стоять в офисе 24/7 — лучше VPS, чтобы не зависеть от Mac.
3. **Камера — SC2336 (родная для P4) или OV5640 (универсальная)?**
   Решим при заказе по доступности на Ali.
4. **GPS — брать сразу или потом?** В офисе работает плохо.
   Имеет смысл если ставить устройство у окна / брать на выезды.
5. **Будем ли таскать устройство на встречи?** Если да — нужен
   power bank через USB-C и GPS обоснован.
