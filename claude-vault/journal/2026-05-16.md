## Решения

- **Кулон, а не готовое и не Telegram.** Полный контроль над прошивкой,
  приватность, выбор LLM, образовательная ценность.
- **Двухстадийная сборка.** Сначала железо без пайки → весь софт.
  Потом упаковка в кулон. Снижает риски для новичка.
- **ESP32-S3** как платформа — Wi-Fi, BLE, AI-ускоритель, дешёвый,
  огромная open-source экосистема.
- **DeepSeek V4** как LLM, **Whisper** как STT, **Yandex SpeechKit**
  как TTS на старте (русский, дёшево).
- **Память — гибрид Obsidian (Markdown) + векторная БД (Chroma)**
  для семантического поиска по прошлым диалогам.

## Дополнение вечера: исследование OpenClaw и выбор железа

### OpenClaw — это реальный проект, не путаница
Поискал в сети «openclaw». Это **личный open-source AI-ассистент**
Питера Штайнбергера, релиз 30 января 2026, к марту — самый звёздный
проект на GitHub (>250k звёзд, обогнал React).
<https://github.com/openclaw/openclaw>

Ключевое:
- TypeScript/Node.js, MIT-лицензия.
- 24 канала общения (Telegram/WhatsApp/iMessage/Signal/...).
- «Eyes and hands»: браузер, файлы, shell.
- Voice Wake + push-to-talk на macOS/iOS/Android.
- Скилы в `~/.openclaw/workspace/skills/` — туда добавляем своё.
- DeepSeek подключается как кастомный OpenAI-совместимый провайдер.

**Известный блок:** Talk Mode STT только en-US (Issue #44688).
Обход — свой Whisper-pipeline (русский Whisper понимает отлично),
текст подаём в OpenClaw как сообщение.

### MimiClaw для ESP32
Прошивка для ESP32-S3, повторяет API OpenClaw на чипе за $5.
<https://github.com/memovai/mimiclaw>
Заменяет собой нашу самописную прошивку (минус 3-4 недели работы).
