## Что хочу обсудить про MCP и скилы

Пользователь спросил, какие MCP и скилы поставить. Мои предварительные предложения (обсудить в следующий раз):

**Сейчас в облаке уже есть:**
- GitHub MCP — работа с PR, issues, кодом ✅
- Базовые инструменты Claude Code (Bash, Read, Write, Edit и т.д.) ✅
- Полезные скилы: `/review`, `/security-review`, `/init`, `/simplify`, `/loop`, `/claude-api` ✅

**Что имеет смысл добавить для бизнеса пользователя:**
- **MCP для PostgreSQL/SQLite** — если будем строить базу клиентов/заявок/лидов в git-форме перерастёт в нормальную БД.
- **Свой кастомный скил** `/daily-tender-report` — единая команда «запусти мониторинг и положи отчёт в ваулт».
- **Свой кастомный скил** `/save-memory` — сохрани текущее состояние работы в vault и закоммить.
- **Свой кастомный скил** `/load-project <name>` — быстро загрузить контекст проекта в начало сессии.

**На Mac (локальный Claude Code) полезно поставить отдельно:**
- Obsidian MCP — чтобы Claude на Mac читал твой личный vault.
- Filesystem MCP — для работы с папками документов клиентов.

Эти настройки нужно делать в `~/.claude/settings.json` на Mac — туда я отсюда не достучусь, но могу написать инструкцию.
