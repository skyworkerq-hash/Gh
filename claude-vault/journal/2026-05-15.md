## Решения

- **Память живёт в `skyworkerq-hash/gh`**, теперь на ветке `main` (создана из feature-ветки). Feature-ветка `claude/ai-capabilities-overview-vWBGX` синхронизирована и остаётся как историческая.
- **Скилы ставятся через SessionStart-хук**, а не коммитом файлов в репо — это даёт автообновление и не раздувает git.
- **`CLAUDE_CODE_REMOTE=true` гейт** — хук работает только в облачных сессиях, на Mac пользователь ставит через `/plugin install`.
- **MCP с API/OAuth не ставим в облако** — только локально на Mac. В облаке только инструменты без авторизации.
- **Подключать скилы по узким триггерам**, не по совпадению темы. Trail of Bits и Understand-Anything — игнорируем (не наш домен).
- **Network access = Full** для среды Default. Принято потому что: DeepSeek API + любые будущие AI-сервисы. Минус: нет белого списка доменов. Можно потом сузить до Custom со списком.
- **`/м` как универсальная команда восстановления контекста.** Работает в любой новой сессии независимо от того, какая ветка стартовая.
