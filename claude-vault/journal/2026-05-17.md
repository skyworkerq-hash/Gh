2. **DevKitC-1 голые кишки — отвергли.** Слишком много пайки для

3. **Домашняя умная колонка LCD-1.85C — рассматривали.** Внешний
   динамик 8Ω 2W, круглый LCD, smart speaker формат. Но пользователь
   склонялся к более мощному.
4. **Финально: офисный AI-хаб на ESP32-P4** — пользователь сам

### Найдено: OpenClaw — реальный проект (углубление)
Подтверждено и углублено вчерашнее открытие: OpenClaw — реальный
open-source AI-ассистент Питера Штайнбергера. 250k+ звёзд на GitHub.
24 канала общения, MIT, скилы в `~/.openclaw/workspace/skills/`.
DeepSeek подключается как кастомный OpenAI-совместимый провайдер.
Talk Mode только en-US (Issue #44688) — обход через свой
Whisper-pipeline.

### MimiClaw не подходит для P4
MimiClaw написан только под ESP32-S3. На P4 не работает.
**Решение:** под P4 пишем прошивку сами на ESP-IDF + ESP-SR
(официальный voice AI стек Espressif). +1-2 месяца разработки.

## Решения

### Форм-фактор: стационарный AI-хаб в офисе
Не носимое. Пользователь работает в офисе, большая часть времени
на одном рабочем месте. Стационарка снимает все wearable-проблемы.

### Чип: ESP32-P4 (а не ESP32-S3)
Платим +1-2 месяца разработки прошивки взамен MimiClaw готового
на S3, но получаем: Wi-Fi 6, RISC-V мощнее, MIPI-CSI для камеры,
GPIO-расширения, on-device AI инструкции.

### Прошивка: ESP-IDF + ESP-SR (свой код)
Не MimiClaw (его на P4 нет), а официальный Espressif стек.
Слой связи с OpenClaw-сервером по WebSocket пишем сами (~2-3 нед).

### Двухэтапный путь к рабочему продукту
- **Этап 1 (1-2 нед):** OpenClaw + DeepSeek + Telegram-канал на Mac.
  Уже можно пользоваться через мессенджер. Без железа.
- **Этап 2 (+1-2 мес после получения P4):** прошивка + перенос в офис.

Так не сидим 3 месяца в молчании — через 2 недели уже что-то
работает в Telegram.

## Заметки на будущее

  с Mac на VPS (Yandex Cloud или Hetzner).

  отдельный заход «что умеет DeepSeek-vision / Doubao-vision».

  если будем распознавать клиентов, нужно их согласие.
