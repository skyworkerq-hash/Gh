---
description: Загрузить всю память пользователя из claude-vault и кратко перессказать на чём остановились
---

Прочитай по порядку и впитай в контекст:

1. `claude-vault/CLAUDE.md` — правила работы с памятью
2. `claude-vault/knowledge/context.md` — кто пользователь, болевые точки, активные проекты
3. `claude-vault/knowledge/preferences.md` — стиль общения, правила скилов/MCP
4. Самый свежий файл в `claude-vault/journal/` (по дате)
5. `claude-vault/projects/*/README.md` для каждого активного проекта
6. Если есть — `claude-vault/secrets/credentials.json` (только смотришь, не цитируешь значения)

Затем ответь пользователю ровно так:

```
✅ Память загружена.

**Кто ты:** [одна строка из context.md]
**Активные проекты:** [список из projects/]
**На чём остановились в прошлый раз:** [2-3 строки из последнего journal]
**Что было запланировано дальше:** [из раздела «Дальше» последнего journal]
```

Без лишних слов — это шаблон, заполни и всё.
